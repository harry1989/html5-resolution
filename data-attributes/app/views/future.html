What else can be done?

  • They can be used in CSS selectors for styling

    
<div>
<span data-login="madhusuc" data-title="PL">Chandra</span>
<span data-login="muralish" data-title="SMT">Harry</span>
<span data-login="jainga" data-title="MT">Gaurav</span>
</div>
<style>
	span[data-title="PL"]{
		background-color: #eaeaea
	}
</style>
    Chandra Harry Gaurav

  • They can be used in CSS expressions, We are not there quite yet to utitlize the data attributes in all CSS expressions, it is supported only for content right now 

    
<div>
<span data-content="madhusuc" data-title="PL">Chandra</span>
<span data-content="muralish" data-title="SMT">Harry</span>
<span data-content="jainga" data-title="MT">Gaurav</span>
</div>

<style>
	span:before{
		content: attr(data-content)
	}
</style>
    Chandra Harry Gaurav

    When they are fully supported you can write code like below..

    
<div>
	<span data-color="blue">I am in blue</span>
	<span data-color="red">I am in blue</span>
</div>
<style>
	span:before{
		color: attr(data-color)
	}
</style>
