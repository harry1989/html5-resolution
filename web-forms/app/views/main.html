HTML5 forms

Web forms has evolved over a time from supporting just POSTing to a server to latest features like ajax support,client side validations, supporting various types of input elements. In this article we take a look into features available to us in HTML5 forms.

Mainly the features can be divided into two parts.

  • Additional input types and client side validations

    • Input types
    • Input attributes like required, pattern, min, max

  • Submitting the forms actions

    • Associating scattered fields with the single form using form attribute
    • Using FormData() instead of having a hidden form
